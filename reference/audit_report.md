## 🔴 CRITICAL CONFLICTS（必须修）

### 1. 核心数值完全不一致 (`tier-constants.ts`)

| 参数 | 世界书 | 我们的代码 | 差异 |
|------|--------|-----------|------|
| 战斗层级系数 | **2.0, 2.8, 4.0, 8.0, 15.0, 35.0, 80.0** | 10, 15, 25, 40, 60, 90, Infinity | 完全不同的数值集 |
| HP 乘数 | **1, 2, 4, 10, 20, 40, 100** | 10, 15, 25, 40, 60, 90, 150 | 完全不同的数值集 |
| MP 乘数 | **1, 2.5, 6, 15, 35, 80, 160** | 5, 8, 15, 25, 40, 60, 100 | 完全不同的数值集 |
| SP 乘数 | 同 MP 乘数 | 5, 8, 12, 20, 30, 45, 75 | 完全不同的数值集 |
| 属性硬上限 | **20（仅 T7 可达）** | T1=20, T7=999 | 天差地别 |
| 每层属性上限 | **[8,10,12,14,16,18,20]** | [20,40,60,80,100,150,999] | 5倍差距 |
| EXP 上限 | **[100,1000,4000,10000,25000,50000,MAX]** | [500,2000,8000,30000,100000,500000,Infinity] | 5倍差距 |

### 2. 骰子系统架构冲突 (`dice.ts`)

- **世界书**: D20 骰池——60 预掷骰，战斗/制作严格顺序取用（防 SL 滥用）
- **代码**: `Math.random()` 实时随机
- **影响**: 战斗协议和制作协议都依赖骰池取骰，当前实现无法对接

### 3. 地理数据几乎全是编造的 (`location-db.ts`)

| 统计 | 说明 |
|------|------|
| ✅ 3% 正确 | 阿斯塔利亚大陆、奥古斯提姆帝国 名称正确 |
| ⚠️ 13% 名称错 | 帝都→艾瑟嘉德、金谷城→金谷河(诺斯加德)、珍珠港→珍珠湾(索伦蒂斯) |
| ❌ 19% 编造 | 北方冰原、东部海域、霜语城、冰晶矿场、深海神殿...全部是咱编的 |
| ➕ 65% 缺失 | 诺斯加德联盟、萨赫拉联邦、赛瑞利亚、翡翠之心、梵尼亚、永夜盟约、瓦伦蒂亚、索伦蒂斯王国、兽族联盟、伯伦斯法环、边陲之国 |

**世界书有 10+ 势力、90+ 地理条目、430+ 拓扑边。我们只有 1 个真实势力 + 2 个编造区域 + 16 节点 + 5 条边。**

## 🟡 MINOR MISMATCHES

### 4. 品质命名体系不一致

| 位置 | 命名 |
|------|------|
| 世界书 | 普通/优良/稀有/史诗/传说/神话/唯一 |
| tier-constants.ts | 优/稀/史/传/神/神/唯一（T5和T6重复） |
| fp-system.ts | poor/normal/good/excellent/masterwork/legendary/mythic |
| CraftActionResult | poor/normal/good/excellent/masterwork（仅5级） |

### 5. 状态效果缺字段 (`types.ts` StatusEffect)

- 世界书要求: 增益/减益/特殊 三分类 → 代码无 `category` 字段
- 世界书要求: 战斗中=回合/脱战=分钟 → 代码 `remainingTime` 无单位上下文

### 6. 纪元名不匹配 (`time-system.ts`)

- 世界书: **复兴纪元**
- 代码默认: **光辉纪元**

### 7. 血脉覆盖不全 (`bloodlines.ts`)

代码 10 种族 vs 世界书 20+ 种族。缺失：翼民、人鱼、光翅妖精、半身人、半人马、霜巨人、构装体、史莱姆、愿灵等

### 8. 好感度范围不对

- 世界书: -100 至 +100（11 级命名区间）
- 代码 `fp-system.ts`: affectionModifier = -50 ~ +100

### 9. 制作系统缺行业/阶段 (`types.ts`)

- `CraftActionRequest` 无 `industry` 字段（锻造/炼金/烹饪/裁缝对应不同属性）
- 无 `stage` 字段（基础加工/半成品/成品，各有独立规则）

### 10. Prompt 模板引用不存在的地点 (`agent-templates.ts`)

模板中引用了 白曜城、诺斯加德 等，但我们的 `DEFAULT_LOCATIONS` 中没有这些地点
