# 世界观叙事上下文参考示例

> **用途**: 当生成与《命定之诗》世界观相关的叙事内容（场景示例、Agent prompt 模板、测试用例等）时，必须参考本示例理解什么是"符合世界观的叙事上下文"。
>
> **核心原则**: 用自然语言描述世界，而非用游戏术语/数值描述世界。

---

## 完整示例：进入铁匠铺

### 场景设定

- **用户输入**: "我推开铁匠铺的门"
- **世界状态**: 光辉纪元001年-05月-24日 16:30，晴朗，春季
- **地点**: 北方-诺斯加德联盟-白曜城-五馆街-铁匠铺
- **剧情**: 第一章"白曜城的新人冒险者"，当前目标"与铁匠公会建立联系"
- **相关记忆**: 铁匠委托（importance:7）、白曜城地理（importance:5）

---

### npc zone 注入内容（NARRATIVE 级别）

## 叙事上下文维度清单

生成世界观叙事内容时，应从以下维度提取信息编织进叙事：

### ✅ 应该考虑并自然地融入叙事

| 维度 | 示例来源 | 叙事中的体现 |
|------|---------|-------------|
| **外貌** | `身材结实的年轻男子，手掌布满老茧` | "你用满是老茧的手推开橡木门" |
| **种族特征** | `矮人` / `精灵` | "身材矮壮的中年矮人" / "银白色的马尾垂在肩头" |
| **背景故事** | `在边境锻造坊长大...拜入老铁匠门下` | 影响角色对特定情境的反应（如见到熟悉的锻造工具会感到亲切） |
| **性格** | `内向、感性...对他人的苦难有天然的同情心` | 影响角色的说话方式和情绪反应 |
| **五维属性** | `力量12 敏捷8` → "结实有力" | "你用结实的臂膀..." / "她轻盈地侧身避开..." |
| **HP状态** | `HP: 85/100` → "手臂的伤口还在隐隐作痛" | 低HP时：动作迟疑、喘息、咬牙坚持 |
| **MP/SP状态** | `SP: 30/125` → "你感到一阵疲惫" | 低SP时：呼吸沉重、动作变慢 |
| **装备风味** | `铁剑 — 朴素但保养良好，剑刃上还留着上次打磨的痕迹` | 提及装备时用外观描述而非数值 |
| **技能** | `重击 — 集中力量进行一次势大力沉的斩击` | 用技能的风格化描述而非 "使用了重击技能" |
| **背包物品** | `治疗药水 ×2 — 散发草药香气的红色液体` | "你从腰间摸出那瓶散发草药香气的红色液体" |
| **金钱** | `50G` / `200G` | "你掏出几枚金币放在柜台上" |
| **状态效果** | `轻微烧伤 — 左手背红肿` | 角色会因疼痛而动作变形或分心 |
| **关系+好感度** | `友好(好感度:45) — 师徒，如同父子` | 影响称呼方式、信任程度、肢体语言 |
| **身份/职业** | `铁匠学徒` / `巡林者` | 影响角色在场景中的行为和专业判断 |
| **当前行为** | `正在锻造` / `靠在门边观察` | 场景开场时角色的即时状态 |
| **位置** | `白曜城-五馆街-铁匠铺` | 环境描写：热浪、金属撞击声、炉火 |
| **时间** | `16:30 下午` / `春季` | 光线、温度、季节氛围 |
| **天气** | `晴朗` | 窗外光线、室内明暗对比 |
| **剧情目标** | `与铁匠公会建立联系` | 铁匠主动提起矿石委托（推动剧情） |
| **相关记忆** | `铁匠委托(importance:7)` | 铁匠记得之前的约定，自然引出 |

### ❌ 不应直接出现在叙事中的内容（破坏世界观沉浸）

| 不应出现 | 错误示例 | 正确做法 |
|---------|---------|---------|
| **装备数值** | "你拔出攻击力+15的铁剑" | "你拔出那把锋利的铁剑" |
| **技能消耗/冷却** | "重击还剩2回合冷却" | （根本不提——冷却由引擎追踪） |
| **物品数值效果** | "喝下恢复20HP的治疗药水" | "辛辣的液体滑过喉咙，伤口的疼痛减轻了几分" |
| **游戏机制术语** | "你获得了50点经验值" | （经验值由引擎结算，叙事中不体现） |
| **五维直接引用** | "你的力量是12，足以举起..." | "你用结实的臂膀稳稳地举起了..." |
| **好感度数值** | "好感度+5" | "老铁匠看你的眼神比刚才柔和了一些" |
| **HP精确数值** | "你还剩85点生命值" | "手臂的伤口还在隐隐作痛，但暂时无碍" |
| **Tier/等级直接引用** | "你是一个T2中坚级别的冒险者" | （身份和职业已经暗示了能力层级） |
| **登神长阶节点名** | "你激活了登神长阶第三节点'剑心'" | （引擎内部追踪，叙事中不过度展开） |

## 使用说明

在以下场景生成内容时，**必须参考本示例**：

1. **编写 Agent prompt 模板** — 特别是 story Agent 的 `fixedSystem` 和 `fixedExamples`
2. **生成设计文档中的场景示例** — 例如此前的 zone_visibility_model.md 中的各 Agent 场景
3. **编写测试用例的 mock 数据** — CharacterState / WorldState 等测试夹具
4. **生成世界书条目内容** — 世界书条目应以自然语言描述世界观，而非游戏机制
5. **编写剧情大纲** — 大纲是叙事蓝图，应以文学化语言而非游戏术语撰写

**检查清单**（生成任何世界观叙事内容后自检）:
- [ ] 有没有出现 `攻击力+X` / `防御力+X` / `+X%` 等数值？
- [ ] 有没有出现 `SP消耗` / `冷却` / `回合` 等战斗机制术语？
- [ ] 有没有出现 `好感度+X` / `经验值` / `HP: X/Y` 等精确数值？
- [ ] 角色行为是否符合其背景、性格、身份、关系？
- [ ] 场景描写是否反映了时间、天气、地点等世界状态？
- [ ] 角色间的互动是否体现了关系亲疏（好感度）？
